Politique de confidentialité

Dernière mise à jour : 1er avril 2026

📌 Résumé en une phrase

Imatra est un logiciel installé localement chez l'agent. L'éditeur ne traite aucune donnée client — toutes les données restent sur la machine du Licencié et ne sont jamais transmises à Imatra ni à un quelconque tiers. Cette politique décrit néanmoins les rares données collectées par le site web public et le serveur de licences.

1. Le logiciel Imatra (installé chez vous)

Le logiciel Imatra que vous installez sur votre ordinateur s'exécute entièrement en local. Tous les documents que vous traitez (CNI, permis, COC, factures, cartes grises, etc.), tous les dossiers, toutes les bases de données, restent sur votre machine.

Imatra n'a aucun accès à ces données. Nous ne sommes pas hébergeur de vos données. Nous ne sommes pas sous-traitant RGPD au sens de l'article 28 du règlement.

Vous restez seul responsable de traitement des données de vos clients, comme avec n'importe quel logiciel professionnel installé localement (Sage, Cegid, EBP, etc.).

2. Sécurité des données locales

Le logiciel Imatra implémente plusieurs mesures de sécurité par défaut sur votre machine :

  • Chiffrement Fernet AES-128 + HMAC SHA-256 de tous les documents stockés
  • Clé de chiffrement générée aléatoirement à l'installation, conservée localement avec permissions restreintes
  • Suppression automatique des données client sensibles après génération du Cerfa (cleanup RGPD intégré)
  • Conservation pour archivage légal limitée au strict nécessaire (nom titulaire + référence dossier, 5 ans)

3. Le site web imatra.fr

Le site public imatra.fr est un site vitrine statique. Il n'utilise aucun cookie de tracking, aucun service d'analytique, aucun outil publicitaire.

Si vous nous écrivez à contact@imatra.fr, nous traitons votre email pour répondre à votre demande puis nous le supprimons quand l'échange est clos.

4. Le serveur de licences

Pour valider votre licence d'utilisation, votre logiciel local contacte une fois par jour licenses.imatra.fr.

Les seules données transmises sont :

  • L'identifiant unique de votre licence (UUID)
  • L'adresse IP publique de votre ordinateur (transmise par votre navigateur réseau, comme pour toute requête HTTP)

Aucune donnée de vos clients, aucun document, aucune information sur les dossiers traités n'est transmise. Le serveur répond uniquement par "valide" ou "révoquée". Vous pouvez utiliser Imatra hors-ligne 30 jours sans aucune connexion à ce serveur.

5. Le serveur de mises à jour des règles

Une fois par jour, votre logiciel vérifie également si une nouvelle version des règles V-XX/C-XX est disponible (évolutions réglementaires SIV, nouveaux tarifs régionaux, nouveaux barèmes malus CO2). Cette vérification est une simple requête HTTP GET anonyme. Aucune donnée client n'est transmise.

6. Données de facturation

Lors de l'achat d'une licence annuelle, nous collectons les informations nécessaires à la facturation : raison sociale, adresse, email, et SIRET. Ces données sont conservées 10 ans conformément à l'obligation comptable (Code de commerce L.123-22) puis supprimées.

7. Vos droits

Pour les seules données vous concernant que nous traitons (email de contact, données de facturation), vous disposez des droits RGPD suivants :

  • Droit d'accès à vos données
  • Droit de rectification
  • Droit à l'effacement
  • Droit à la limitation du traitement
  • Droit à la portabilité
  • Droit d'opposition

Pour exercer ces droits : contact@imatra.fr. Nous répondons sous 30 jours maximum. En cas de désaccord, vous pouvez introduire une réclamation auprès de la CNIL (cnil.fr).

8. Données de vos clients

Important : les données des clients de l'agent (les particuliers ou personnes morales pour lesquels vous effectuez des cartes grises) ne sont jamais traitées par Imatra. Elles restent sur votre machine, sous votre responsabilité de traitement, conformément au RGPD. C'est à vous de respecter les obligations RGPD vis-à-vis de vos propres clients (information, consentement, durées de conservation, droit d'accès, etc.).
