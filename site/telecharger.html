Installation pas à pas

  1. 1. Décompresser l'archive

    Extrayez le fichier ZIP dans un dossier dédié, par exemple C:\ImatraPro sur Windows ou ~/ImatraPro sur Mac/Linux.

  2. 2. Lancer le script d'installation

    Double-cliquez sur le script correspondant à votre système :

    Windows : install.bat
    macOS / Linux : bash install.sh

    Le script vérifie que Docker est installé, build l'image (5 à 15 minutes au premier lancement) puis démarre Imatra automatiquement.

  3. 3. Ouvrir le logiciel dans le navigateur

    Une fois le démarrage terminé, ouvrez votre navigateur web et allez à :

    http://localhost:8001

  4. 4. Configurer votre profil

    Dans Paramètres → Profil de l'agent, renseignez votre raison sociale, adresse, numéro d'habilitation SIV. Uploadez votre cachet et votre signature numérique.

  5. 5. Commencer à traiter des dossiers

    Dans le tableau de bord, glissez un email reçu d'un client. Le système extrait les pièces, fait l'OCR, lance les vérifications et propose le diagnostic. Vous êtes opérationnel.
